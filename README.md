# 🧮 Calculator App (Vanilla JS)

A calculator application built with **Vanilla JavaScript**, focusing on state management, DOM manipulation, and event-driven interactions.

## Features

* Addition, subtraction, multiplication and division
* Percentage calculation
* Dynamic display rendering
* Operator handling
* Calculator reset/clean functionality
* State-based application flow

## Concepts Applied

* State-driven architecture
* DOM manipulation
* Event handling
* Conditional rendering
* Array iteration (`forEach`)
* Data attributes (`dataset`)
* Separation of concerns
* Dynamic UI updates

## Tech Stack

* HTML
* CSS
* JavaScript (ES6+)

## Application Structure

The calculator uses a global state object to manage:

* Current value
* Previous value
* Active operator

Rendering is dynamically updated based on the current application state.

## How to Run

1. Clone the repository
2. Open `index.html` in your browser

## 📌 Notes

This project was built as a learning exercise to practice JavaScript fundamentals, state management, and event-driven programming before moving to frameworks like React.
